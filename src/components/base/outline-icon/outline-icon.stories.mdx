# Using Outline Icon Component

## Outline Icon Attributes

- `color` string - Color of icon. Default: `currentColor`
- `decorative` boolean - Sets aria-hidden attribute to hide from screen-readers. See accessible icons section below
- `name` string (Required) - Icon name as defined above
- `mobile-size` string - Height and width of icon on mobile screens. Default: `16px`
- `size` string - Height and width of icon on screens larger than mobile. Default: `32px`

## Using Generate Icon Script

- Place project icon svgs in `src/components/base/outline-icon/svgs/` in one or seperate folders. (icon filenames must be kebab-case)
- Run `yarn run generate.icons`, this will generate `outline-icons/iconList.ts` and `outline-icons/generated-svgs/<folderNameIcons>`
- Check files for discrepencies and run Prettier to format files
- Anytime svgs are added, removed, or modified then run the last previous two steps again

## Accessible Icons

### Icon is Decorative

If the icon is solely decorative, add decorative to the icon component `<outline-icon decorative></outline-icon>`

### Icon is Wrapped by Link

If the link has visible text then add the `decorative` attribute to `outline-icon` and the text will serve as the label.
If the SVG is the link wrapped in an `<a>` tag, then give it an accessible label `<a href="/" aria-label="I am a link to xyz website"><outline-icon decorative></outline-icon></a>`
